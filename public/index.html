Timestamp Microservise description
  • Input date in format:
    • January 1, 2016
    • 1450137600
  • Returns null values if parameter is not provided or input format is invalid
  • Sample queries:
    • https://timestamp-microservice-rfprod.c9users.io/December%2015,%202015
    • https://timestamp-microservice-rfprod.c9users.io/1450137600
    • https://timestamp-microservice-rfprod.c9users.io/this%20is%20not%20a%20valid%20date
  • Output examples:
    • { "unix": 1450137600, "natural": "December 15, 2015" }
    • { "unix": null, "natural": null }
